# Web Color Tools
A website with color tools commonly used in web development.

- Note: Not all features are implemented yet
## Random Color
- Get a random color
- Save colors for later
## Color Palettes
- Choose a palette type
    - Monochromatic
    - Analogous 
    - Complementary 
    - Split-Complementary
    - Triadic
    - Tetradic
    - Square
    - Analogous Complementary
- Save color palettes for later
## Gradient Maker
- Linear gradient
    - CSS code snippet
## Accessibility
- Contrast Checker for WCAG 2.1 AA and 2.1 AAA
    - Standard text
    - Large text
    - Graphics and UI
