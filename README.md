# Web Color Tools
A website with color tools commonly used in web development. Version 2.

## Random Color
- Get a random color
- Save colors for later
- Favorite colors
## Color Palettes
- Choose a palette type
    - Monochromatic
    - Analogous 
    - Complementary 
    - Split-Complementary
    - Triadic
    - Tetradic
    - Square
- Save color palettes for later
- Favorite palettes
## Gradient Maker
- Gradient type selection
  - Linear
  - Radial
  - Conic
- CSS code snippet
- Save gradients for later
## Accessibility
- Contrast Checker for WCAG 2.1 AA and 2.1 AAA
    - Standard text
    - Large text
    - Graphics and UI
